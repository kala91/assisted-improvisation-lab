# Assisted Improvisation Lab

Static website and working archive for a digitally assisted improvisation theatre method.

The project collects:

- a concise method manifesto
- game recipes and facilitation formats
- prototype notes, including Elopeli as one case
- process logs from the grant-funded development work
- workshop and visit descriptions

## Operating Model

Aleksi provides high-level direction, field notes, photos, screenshots, rough drafts, and project reflections.
The public site edits this material into a structured method library, prototype archive, and workshop-facing presentation.

## Local Preview

Open `index.html` in a browser. The site is intentionally plain static HTML/CSS for GitHub Pages.

## Content Intake

Draft material can arrive through Telegram for short notes and through Nextcloud for images, screenshots, PDFs, and longer text drafts.
